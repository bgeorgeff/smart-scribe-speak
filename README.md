# Learn Anything

An educational reading tool that helps users learn to read through interactive syllable-based word practice, speech-to-text exercises, and AI-generated educational content.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn-ui
- Supabase (auth, database, edge functions)

## Development

```sh
npm install
npm run dev
```

## Deployment

The app is hosted on [Replit](https://replit.com). Supabase Edge Functions are deployed via the Supabase CLI.
